# 1.3 建设内容

本项目建设内容包括沙尘动态预测AI模型训练、沙尘监测预警示范服务场景应用平台、三领域智能体服务接入和微信小程序四个部分。

**沙尘动态预测AI模型训练。** 构建适用于宁夏区域的沙尘动态预测AI模型，融合物理机理与深度学习方法，基于历史沙尘天气过程、气象观测、数值模式、卫星遥感和大气环境监测等多源数据建立训练数据集，实现未来24小时以上沙尘浓度或风险空间分布的精细化预报。配套建立模型检验评估流程，通过历史个例回溯、实况监测和遥感判识对模型性能进行持续评估。

**沙尘监测预警示范服务场景应用平台。** 建设面向专业用户的B/S架构平台，集成沙尘实况监测、预报预警、产品制作、GIS空间分析和系统管理功能。平台采用微服务架构和流批一体化框架，支持7×24小时不间断运行，界面响应不超过3秒、前端交互不超过2秒，满足100用户50并发的性能要求，具备可扩展和易维护特性。

**三领域智能体服务接入。** 通过标准化API网关接入宁夏移动建设的交通、人体健康、旅游三领域智能体，完成参数组织、调用编排、结果解析和融合展示。本项目负责接口适配和服务融合，不重复建设三领域智能体。行业系统可通过API接口查询区域风险、道路通行风险、景区影响等级，并订阅预警信息。

**微信小程序。** 开发面向公众的微信小程序，提供实时能见度地图、当前位置沙尘风险等级、未来趋势和预警提醒、个性化防护建议，以及关注区域订阅和消息推送功能。

上述四部分内容相互支撑：模型训练提供核心预测能力，平台提供业务运营环境，智能体接入扩展场景分析深度，小程序触达公众用户。项目整体在合同签订后24个月内完成验收。
